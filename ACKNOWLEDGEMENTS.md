# Acknowledgements

CodeStable Compact is an original implementation informed by the following public projects:

- **CodeStable** — https://github.com/liuzhengdongfortest/CodeStable  
  Inspiration: software-lifecycle entities, human ownership, project-local durable state, and requirements/decisions/features/issues as first-class engineering records. License in the referenced repository: MIT.

- **Trellis** — https://github.com/mindfold-ai/Trellis  
  Inspiration: task-centered state, resumable phases, compact indexes, applicable-spec loading, and promotion of durable findings from completed work. License in the referenced repository: AGPL-3.0.

- **ponytail** — https://github.com/DietrichGebert/ponytail  
  Inspiration: prefer deletion/reuse/stdlib, solve the root cause, avoid speculative abstractions and dependencies, and keep host adapters separate from canonical skills. License in the referenced repository: MIT.

No source code or prose from these projects is vendored here. The design is deliberately re-expressed and independently implemented so that this repository can use the MIT license.
